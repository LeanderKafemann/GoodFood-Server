# GoodFood Server

server for GoodFood dbs.

More about [GoodFood](https://github.com/LeanderKafemann/GoodFood)

Last updated for GoodFood <b>1.1.0</b>

## Installation

1. Download this repository on your server.
2. Make sure you have at least Python 3.11 installed.
3. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```
4. Execute setup.py to initialize the password (numeric), the database and more.
5. Start the server:
   ```bash
   python server.py
   ```
   Or double click on server.py if your system is configured to run Python files with Python.
